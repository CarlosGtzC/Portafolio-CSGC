Bienvenido a mi portafolio de proyectos de Análisis de Datos

Este repositorio reúne proyectos desarrollados como parte de mi transición profesional hacia el rol de Data Analyst, donde aplico técnicas de análisis, limpieza y visualización de datos para resolver problemas reales y apoyar la toma de decisiones.

A lo largo de los proyectos utilizo herramientas como Python (pandas, numpy, matplotlib, seaborn), SQL y Excel, enfocándome en transformar datos crudos en insights accionables. Cada proyecto incluye el contexto del problema, el proceso de análisis y las conclusiones obtenidas.

Este portafolio refleja mi enfoque analítico, mi orientación a la resolución de problemas y mi interés por generar valor a partir de los datos.
